Тестовый пример создания клиент-серверного приложения "Калькулятор".
Для создания данного приложения был использован Framework QT, так как он обладает большим функционалом для работы с network. Формат передачи данных JSON.

Запуск программы:
1.Запустить сервер (Server/release/myServer.exe)
2.Запустить клиент (Client/release/mycalculator_client.exe)
